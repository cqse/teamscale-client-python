teamscale_client.utils.auto_str(cls)[source]¶

Annotation that provides a default __str__ method for objects.

Example

Annotating a class with the @auto_str annotation:

>>> @auto_str
    class Dummy(object):
        def __init__(self, a):
            self.a = a

>>> sample = Dummy("test")
>>> print(sample)
Dummy(a=test)
